# BMRK

Benchmarks for Ruby.

[`fast-ruby`](https://github.com/fastruby/fast-ruby) has way more benchmarks.

You need `ruby` and `bundler` to run this.

Clone repository:

```
git clone https://github.com/endormi/bmrk.git && cd bmrk
```

Install dependencies:

```
bundle install
```

Run all benchmarks:

```
rake
```

Run files separately:

```
ruby <filename>.rb
```
